GQ API（以下简称"我们"）提供统一的 AI API 网关服务，允许用户通过单一接口访问多种人工智能模型。

## 2. 账户注册

2.1 您在注册时应当提供真实、准确、完整的个人或企业信息。

2.2 您应当妥善保管账户信息，对账户下的所有行为负责。

2.3 如发现账户被非法使用，应立即通知我们。

3.1 您应遵守所在国家/地区的法律法规。

3.3 我们保留在不通知的情况下暂停或终止违规账户的权利。
