# 手机 Chatbox 配置

Chatbox 是一款跨平台的 AI 客户端，支持在手机上方便地使用 GQ API 服务。

## 下载安装

### iOS

从 App Store 搜索 **Chatbox AI** 下载安装。

### Android

从 Google Play 或应用官网下载 APK。

## 配置步骤

### 1. 打开设置

启动 Chatbox 后，点击左上角 **设置** 图标。

### 2. 配置 API 连接

找到 **API 设置** 部分：

| 配置项 | 值 |
|--------|-----|
| API Provider | 选择 **Custom (OpenAI Compatible)** |
| API Base URL | `https://api.gqchat.cn/v1` |
| API Key | 输入您的 GQ API API Key |

### 3. 选择模型

在模型列表中，您可以选择：

- `claude-sonnet-4` — 推荐
- `gpt-4o`
- `gemini-2.5-flash`

也可以手动输入模型名称来使用其他模型。

### 4. 保存并开始聊天

返回主界面，开始与 AI 对话！

## 移动端优化建议

### 流式输出

确保开启 **流式输出（Stream）** 选项，获得更好的打字机效果体验。

### 语音输入

Chatbox 支持语音转文字功能，长按输入框即可使用语音输入。

### 导出对话

重要对话可以导出为 Markdown 或 PDF 格式保存。

## 同步配置

如果您同时在桌面端使用 Chatbox，可以通过云端同步配置：

1. 注册 Chatbox 账户
2. 在设置中开启 **云同步**
3. 手机和电脑的配置将自动同步
