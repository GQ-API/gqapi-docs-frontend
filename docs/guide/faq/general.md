# 通用问题

## 账户与计费

### Q: 如何注册 GQ API 账户？

访问 [gq-api.com](https://api.gqchat.cn/console) 点击注册即可。支持邮箱注册和第三方 OAuth 登录。

### Q: 有免费额度吗？

新用户注册后可获得一定的免费体验额度，用于测试 API 功能。

### Q: 如何充值？

登录控制台 → 充值中心 → 选择支付方式完成充值。

支持的支付方式：
- 支付宝 / 微信支付
- 信用卡（Visa/Mastercard）
- 加密货币（USDT）

### Q: 费用如何计算？

按实际消耗的 **Token 数量** 计算：

> 费用 = 输入 Token × 输入单价 + 输出 Token × 输出单价

不同模型单价不同，详见[定价页面](https://api.gqchat.cn/console/pricing)。

## API 使用

### Q: API Base URL 是什么？

```
https://api.gqchat.cn/v1
```

### Q: 支持流式输出吗？

完全支持。设置 `"stream": true` 即可启用 SSE 流式响应。

### Q: 有请求频率限制吗？

有。不同套餐有不同的 RPM（每分钟请求数）限制：
- 基础版：60 RPM
- 专业版：300 RPM
- 企业版：自定义

### Q: 请求超时时间是多少？

默认超时 **180 秒**。流式连接保持活跃状态。

## 安全与隐私

### Q: 我的数据安全吗？

::: info 隐私承诺
- 不存储您的提示词内容
- 不使用您的数据训练模型
- 所有传输均使用 TLS 加密
- 符合 GDPR 数据保护要求
:::

### Q: API Key 泄露了怎么办？

立即在控制台**吊销**该 Key 并创建新的：

控制台 → 令牌管理 → 吊销 → 新建令牌

## 技术支持

### Q: 如何获取技术支持？

- **文档**：您正在阅读的就是完整文档
- **社区**：加入 QQ群 或 Discord
- **工单**：发送邮件至 support@gqchat.cn

### Q: SLA 服务等级协议是什么？

企业版用户享有 **99.9%** 的可用性 SLA 保障。
