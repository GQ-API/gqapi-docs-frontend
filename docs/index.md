## 为什么选择 GQ API？

GQ API 是一个统一的 AI API 网关，让您通过一个 **API 密钥**、一个 **接入点** 即可访问所有主流 AI 模型 —— **Claude**、**GPT**、**Gemini**、**DeepSeek** 等等。

无需管理多个账户，无需分别计费，即插即用。

## 适用人群

- **开发者**：在个人项目中快速集成多种 AI 能力
- **研究人员**：灵活对比和测试不同模型的表现
- **团队协作**：统一管理 API 调用，简化权限控制
- **企业用户**：生产级工作负载的稳定支撑
