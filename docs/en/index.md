## Why GQ API?

GQ API is a unified AI API gateway that lets you access all major AI models through **one API key** and **one endpoint** — **Claude**, **GPT**, **Gemini**, **DeepSeek**, and more.

No need to manage multiple accounts or separate billing. Plug and play.

## Who Is This For?

- **Developers**: Quickly integrate multiple AI capabilities into personal projects
- **Researchers**: Flexibly compare and test different model performances
- **Teams**: Unified API call management with simplified permission control
- **Enterprise**: Production-grade stability for mission-critical workloads
