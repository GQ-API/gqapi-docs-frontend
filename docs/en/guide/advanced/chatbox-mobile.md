# Mobile Chatbox Setup

Chatbox is a cross-platform AI client that lets you easily use GQ API on mobile devices.

## Download & Install

### iOS

Search **Chatbox AI** on the App Store.

### Android

Download APK from Google Play or the official website.

## Configuration Steps

### 1. Open Settings

Launch Chatbox and tap the **Settings** icon (top-left).

### 2. Configure API Connection

Find **API Settings**:

| Setting | Value |
|---------|-------|
| API Provider | Select **Custom (OpenAI Compatible)** |
| API Base URL | `https://api.gqchat.cn/v1` |
| API Key | Enter your GQ API Key |

### 3. Select Model

Choose from the list:

- `claude-sonnet-4` — Recommended
- `gpt-4o`
- `gemini-2.5-flash`

Or manually enter any supported model name.

### 4. Save & Start Chatting

Return to the main screen and start chatting!

## Mobile Optimization Tips

### Streaming Output

Enable **Stream** mode for better typewriter effect experience.

### Voice Input

Long-press the input field to use voice-to-text.

### Export Conversations

Export important chats as Markdown or PDF.
