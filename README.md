## ✨ 特性

- 🌍 **中英双语** — 完整的中文 / English 文档支持，一键切换
- 📚 **完整教程** — CLI 配置、模型介绍、绘图、进阶玩法全覆盖
- 📱 **响应式设计** — 适配桌面端和移动端
- 🌙 **深色模式** — 支持亮色/暗色主题切换
- 🔍 **本地搜索** — 内置全文搜索功能

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
